Getting Started

This UI Component Library provides a collection of ready-to-use UI components that you can easily copy and paste into your projects.

Each component is designed to be standalone and does not require any external dependencies or npm packages.

How to Use

  1. Browse the component library to find the component you need.
  2. Click on a component to view its details, preview, and code.
  3. Customize the component using the available options.
  4. Copy the HTML code and paste it into your project.
  5. Add your own styles or modify the existing ones to match your design.

Customization

All components use CSS classes for styling. You can customize the appearance by modifying these classes.

For components with variants, you can select different styles from the customization panel.

Accessibility

All components are designed with accessibility in mind. They include proper ARIA attributes and follow best practices for keyboard navigation and screen reader support.

Browser Support

The components are compatible with all modern browsers, including Chrome, Firefox, Safari, and Edge.

Built with HTML, CSS, and vanilla JavaScript.
